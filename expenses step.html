Complete Expense Management Features

NEW: Drag & Drop Ordering:

  • • Drag Handle: ⋮⋮ icon on left side for clear drag affordance
  • • Visual Feedback: Handle changes color on hover
  • • Cursor Changes: Grab cursor indicates draggable items
  • • Custom Ordering: Users can arrange expenses by priority/preference

NEW: Hide from Budget Feature:

  • • Hide/Show Toggle: Button to exclude expenses from budget totals
  • • Visual Distinction: Hidden expenses are grayed out with dashed border
  • • "HIDDEN" Badge: Clear indicator of hidden status
  • • Category Totals: Show count of hidden expenses
  • • Budget Exclusion: Hidden expenses don't count toward allocation

Bulk Add Feature:

  • • Quick Add Multiple Button: Alongside regular add button
  • • Text Area Input: Multiple expense names, one per line
  • • Smart Counting: Shows "Create X Expense Items"
  • • Stub Creation: Creates expenses that need completion

Use Cases for Hide Feature:

  • • Seasonal Expenses: Keep Christmas gifts visible but not active year-round
  • • Considering Options: Track potential expenses without committing
  • • Temporary Suspension: Gym membership on hold but might restart
  • • Budget Testing: See impact of removing expenses temporarily
  • • Future Planning: Expenses planned for later addition

Visual State System:

  • • Complete: Normal styling, counted in budget
  • • Incomplete (Stub): Orange border, "Fill Out" action
  • • Non-Monthly: Blue left border with frequency badge
  • • Hidden: Grayed out, dashed border, not counted
  • • Editing: Green border when form is expanded

Smart Category Totals:

  • • Active Amount: Only counts visible/active expenses
  • • Hidden Count: Shows "(2 hidden)" indicator
  • • Real-time Updates: Totals change as items are hidden/shown
  • • Budget Accuracy: Ensures allocations reflect reality
